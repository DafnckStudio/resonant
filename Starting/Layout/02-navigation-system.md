## Navigation Hierarchy

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                         NAVIGATION ARCHITECTURE                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  LEVEL 1: PRIMARY NAVIGATION (Sidebar)                                          │
│  ├── 🏠 Dashboard                    → /dashboard                               │
│  ├── 📊 Workflows                    → /workflows                               │
│  ├── 📁 Templates                    → /templates                               │
│  ├── ⚡ Executions                   → /executions                              │
│  ├── 🛒 Marketplace                  → /marketplace                             │
│  └── ⚙️ Settings                     → /settings                                │
│                                                                                  │
│  LEVEL 2: SECONDARY NAVIGATION (Page Tabs)                                      │
│  └── e.g., Settings                                                             │
│      ├── Profile                     → /settings/profile                        │
│      ├── Workspace                   → /settings/workspace                      │
│      ├── Team                        → /settings/team                           │
│      ├── Billing                     → /settings/billing                        │
│      ├── AI Providers                → /settings/ai-providers                   │
│      └── Integrations                → /settings/integrations                   │
│                                                                                  │
│  LEVEL 3: TERTIARY NAVIGATION (Breadcrumbs)                                     │
│  └── e.g., Workflows > Blog Generator > Settings                                │
│                                                                                  │
│  LEVEL 4: CONTEXTUAL NAVIGATION (In-page)                                       │
│  └── e.g., Anchor links, quick actions                                          │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```
