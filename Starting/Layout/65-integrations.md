## Responsive Behavior

### Tablet (768px - 1024px)
- Settings navigation becomes horizontal tabs at top
- Integration cards grid: 2 columns
- Modals remain full-width with padding

### Mobile (< 768px)
- Settings navigation becomes list view with back button
- Integration cards stack vertically
- Modals become full-screen sheets
- API keys show abbreviated format with expand option
- Webhook URLs show truncated with copy button
