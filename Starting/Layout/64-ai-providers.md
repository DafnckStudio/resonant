# 64 - AI Providers Settings

> API key management for AI providers, model configuration, and usage monitoring.

---

## Page Purpose

The AI providers settings page allows users to configure their own API keys for various AI providers (OpenAI, Anthropic, Google, etc.), manage model preferences, set default configurations, and monitor usage across providers. This enables BYOK (Bring Your Own Key) functionality and cost control.
