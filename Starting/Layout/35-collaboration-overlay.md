### Cursor Movement

```css
.remote-cursor {
  transition: transform 50ms linear;
}

.cursor-label {
  transition: opacity 150ms ease-out;
}

.cursor-label.idle {
  opacity: 0.6;
}
```

### Selection Indicator

```css
.collaborator-selection {
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0%, 100% { box-shadow: 0 0 0 0 var(--color); }
  50% { box-shadow: 0 0 0 4px var(--color-transparent); }
}
```

### User Join/Leave

```css
.presence-toast {
  animation: slideIn 300ms ease-out;
}

@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
```

```css
.activity-item {
```
