## Animations

### Timeline Entry Expand

```css
.version-details {
  animation: expandDown 200ms ease-out;
}

@keyframes expandDown {
  from {
    opacity: 0;
    max-height: 0;
  }
  to {
    opacity: 1;
    max-height: 200px;
  }
}
```

### Diff Highlight

```css
.diff-added {
  background-color: var(--green-100);
  animation: highlight 300ms ease-out;
}

.diff-removed {
  background-color: var(--red-100);
  animation: highlight 300ms ease-out;
}

@keyframes highlight {
  0% { opacity: 0.5; }
  100% { opacity: 1; }
}
```

### Restore Animation

```css
.restoring {
  animation: pulse 1s infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
```
