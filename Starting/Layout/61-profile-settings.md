# 61 - Profile Settings

> User profile and account settings page with personal information, preferences, and security options.

---

## Page Purpose

The profile settings page allows users to manage their personal account information, customize their experience preferences, and configure security settings. This is the primary location for individual user configuration separate from workspace-level settings.
